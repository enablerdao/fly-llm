Easy to Use

Simple API compatible with OpenAI clients

Automatic Model Selection

curl -X POST https://{{ host }}/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "auto",
    "messages": [{"role": "user", "content": "What is the capital of Japan?"}]
  }'

Just specify "model": "auto" and our system will select the best model for your query.

Specific Model Selection

curl -X POST https://{{ host }}/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Explain quantum computing"}]
  }'

Or specify exactly which model you want to use for complete control.
